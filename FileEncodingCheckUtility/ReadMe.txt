Follow below Steps to run the encoding of files
Steps:
		1.Specify the folder path in key "InputFolderPath" where the files are kept for which we need to check encoding. Also specify multiple path separating comma.
		2.Specify encoding in key "validEncodingtype" which will be checked against files encoding. Multiple encoding can be specified and separated by comma.
		3.Specify the output log file path in key "logFilePath". This file will have the list of files whose encoding does not match with the specified encodings.
		4.Run the application.
		5.check log file for the output.
